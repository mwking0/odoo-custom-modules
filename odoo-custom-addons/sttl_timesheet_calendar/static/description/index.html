Odoo 15 Timesheet Calendar

OVERVIEW:
Most of Businesses often uses traditional methods to fill the project timesheet, such as Excel, where mentioning start time and end time of any task, tend to be clunky and inefficient.

Nowadays Businesses prefer drag and drop feature to optimize all of their performance, where Calendar timesheet App provides you various benefits:

Benefits:

Time Efficient

Easy Access

Increased Productivity

How it works--

1. This Application provides a calendar view that lets you to fill the project timesheet by drag and select feature.

2. New Event screen will pop, where you have to add the description of task. Hit the create button.

3. A new pop-up screen lets you to fill more details like Project, Task, and Description. Hit the Save button to save. Later on you can do changes or modification until Lock time.(Lock Timesheet App)

4. This way, when you view the MyTimesheet, you can view all My timesheet filled. You view it by various group by like Date, Project, and Task

Supported: v14 15 16.
